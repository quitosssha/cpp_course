## Лабораторная работа 2

**Тема:** Работа с матричным представлением данных. Обработка матриц.

### Цель работы

Приобретение навыков работы с матричным представлением данных и разработки функций обработки матриц на языке C++.

### Задание

Используя заготовку для ввода и вывода квадратной матрицы размера N×N, написать функции, выполняющие следующие операции:

1. Нахождение минимума матрицы.  
2. Нахождение максимума матрицы.  
3. Нахождение максимума нижнетреугольной части матрицы.  
4. Нахождение максимума верхнетреугольной части матрицы.  
5. Нахождение минимума нижнетреугольной части матрицы.  
6. Нахождение минимума верхнетреугольной части матрицы.  
7. Нахождение минимума главной диагонали матрицы.  
8. Нахождение максимума главной диагонали матрицы.  
9. Нахождение минимума второстепенной диагонали матрицы.  
10. Нахождение максимума второстепенной диагонали матрицы.  
11. Вычисление среднеарифметического значения элементов матрицы.  
12. Вычисление среднеарифметического значения элементов нижнетреугольной части матрицы.  
13. Вычисление среднеарифметического значения элементов верхнетреугольной части матрицы.  
14. Вычисление сумм строк матрицы.  
15. Вычисление сумм столбцов матрицы.  
16. Нахождение минимальных значений строк.  
17. Нахождение минимальных значений столбцов.  
18. Нахождение максимальных значений строк.  
19. Нахождение максимальных значений столбцов.  
20. Вычисление среднеарифметических значений строк.  
21. Вычисление среднеарифметических значений столбцов.  
22. Вычисление сумм нижней и верхней треугольных частей матрицы.  
23. Нахождение элемента, наиболее близкого по значению к среднеарифметическому значению элементов матрицы.

### Заготовочная программа

В качестве основы использовать заготовочную функцию для матричных операций (инициализация и вывод матрицы):

```cpp
#include <iostream.h>
#include <iomanip.h>
#include <stdlib.h>
#include <conio.h>
#define N 5

void main()
{
    float m[N][N];
    int i, j;
    for(i = 0; i < N; i++)
        for(j = 0; j < N; j++) 
            m[i][j] = rand() / 10.0;

    for(i = 0; i < N; i++)
    {
        for(j = 0; j < N; j++)
            cout << setw(8) << setprecision(5) << m[i][j];
        cout << endl;
    }
    getch();
}
```
