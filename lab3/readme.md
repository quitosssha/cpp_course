## Лабораторная работа 3

**Тема:** Методы упорядочения числовых данных. Алгоритмы сортировки в C++.

### Цель работы

Освоение основных методов упорядочения числовых данных и знакомство с реализацией различных алгоритмов сортировки средствами языка C++.

### Задание

1. Разобраться, как работает предложенная программа сортировки массива методом «мини-макса».

2. Написать программы, сортирующие одномерный массив следующими методами:
   - методом «пузырька»;
   - методом «быстрой сортировки» (quicksort).

   Ответить на вопрос: можно ли такие функции использовать для работы с многомерным массивом?

3. Для массива целых значений выполнить сортировку:
   - по возрастанию чётных элементов;
   - по убыванию нечётных элементов.

4. Отсортировать массив по возрастанию на интервале индексов от N1 до N2.

5. Отсортировать массив по убыванию на интервале индексов от N1 до N2.

### Пример программы сортировки методом «мини-макса»

```cpp
#include <iostream.h>
#include <conio.h>

void main(void)
{
    int mas[] = {2,5,8,1,4,6,3,-5,-9,13,0,4,9};
    // текущие индексы мин. и макс. элементов
    int imin, imax;
    // вычисление количества элементов
    int n = sizeof(mas) / sizeof(int);
    int i;

    // установка начальных индексов для мин и макс
    imin = i = 0; 
    imax = i = 0;

    clrscr();
    for(i = 0; i < n; i++)
        cout << mas[i] << ' ';
    cout << endl;

    for(i = 0; i < n - 1; i++)
    {
        imin = i;
        for(int j = i + 1; j < n; j++)
            // нахождение минимального элемента
            if (mas[j] < mas[imin]) 
                imin = j;
        int t = mas[i];
        mas[i] = mas[imin];
        mas[imin] = t;
    }

    for(i = 0; i < n; i++)
        cout << mas[i] << ' ';
    cout << endl;
}
```


## Ответы на вопросы
1. Можно ли использовать функции сортировки одномерного массива для многомерного?

Можно, если «развернуть» двумерный массив в одномерный (линейно пробежать по элементам и сортировать их как один массив). В чистом виде функции выше работают только с одномерным массивом int[], для матрицы нужно либо менять сигнатуру (например, сортировать строки по отдельности), либо передавать указатель на первый элемент и размер как на линейный массив.
